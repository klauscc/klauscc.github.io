Feng Cheng

  • I am currently a Master student at Shanghai Jiao Tong Univerisity, and I'm expected to graduate in March, 2019. My research advisor is Dr. Shi-Lin Wang.
  • I received my Bachelor's degree in Shanghai Jiao Tong University.
  • My primary research interests include spatial-temporal sequence learning, lip content recognition and authentication.

  • I am skilled in c/c++, python, java. Beyond these, I can quickly transfer to a new language or new area if needed (they are almost the same except for the rules). I have written several projects in several languages actually:

    • A c library jpeg_toolbox_python with both c API and python API. The library is to extract the DCT coefficient of the JPEG images.
    • My implementation of paper LipNet: End-to-End Sentence-level Lipreading using Keras(Tensorflow backend). Interestingly, this project got 16 stars and 7 forks.
    • An Android app which implements the idea proposed in my paperVisual speaker authentication with random prompt texts by a dual-task CNN framework. We perform authentication with liveness detection only with visual information, specifically, the lip region. The code has not been open-sourced due to business limits. click here for video demo
